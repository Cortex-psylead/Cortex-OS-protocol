The **Cortex Protocol** is an open-source framework designed to protect the last frontier of human privacy: the mind. It provides a secure layer between biometric sensors and Artificial Intelligence (Acolytes), ensuring that your neural data remains yours—mathematically, clinically, and legally.

---

## 🛡️ Milestone 0: The Cognitive Shield (Completed)
We have successfully implemented the core protection engine. The **Sovereignty Abstraction Layer (SAL)** is now functional.

**Key Technical Features:**
* **Sensor Hardening:** Automatic rejection of uncertified or low-fidelity hardware.
* **Anonymous Tensor Transformation:** Irreversible conversion of raw brainwaves into inert mathematical objects using HMAC-SHA256 and dimensionality reduction.
* **Clinical Drift Index (CDI):** A proactive "immune system" that monitors for pathological stress or behavioral manipulation by guest AIs.
* **Secure Ephemeral Memory:** Automatic memory overwriting to ensure raw data never persists.

> Check the implementation at: `src/sal/cognitive_shield.py`
