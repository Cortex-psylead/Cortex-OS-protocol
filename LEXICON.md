 GLOSSARY

## 1. Proprietary Concepts (Cortex Original)

* **White Branch** [ES: Rama Blanca / Tribunal de Ética]: The sovereign governing body and ethics tribunal of the protocol. It is composed exclusively of mental health and neuroscience professionals.
* **Acolyte** [ES: Acólito]: Any external AI entity, agent, or model interacting with the user's data stream through the protocol.
* **Clinical Drift Index (CDI)** [ES: Índice de Deriva Clínica]: Cortex’s proprietary metric that calculates the statistical deviation between current neurophysiological states and the Baseline.
* **Sovereignty Abstraction Layer (SAL)** [ES: Capa de Abstracción de Soberanía]: The technical core that transforms raw signals into irreversible, anonymous tensors.
* **Cognitive Shield** [ES: Escudo Cognitivo]: The active state of the protocol where all defense barriers (SAL + CDI) are engaged.

## 2. Integrated Scientific Foundations

* **Polyvagal Theory** [ES: Teoría Polivagal]: *Source: Dr. Stephen Porges.* Used to categorize autonomic nervous system states.
* **Window of Tolerance** [ES: Ventana de Tolerancia]: *Source: Dr. Dan Siegel.* The range of arousal where a person operates effectively.
* **Tensor** [ES: Tensor]: Multidimensional mathematical object for data representation.
* **Homeostasis** [ES: Homeostasis]: The dynamic internal balance the protocol seeks to preserve.

## 3. Protocol Status Taxonomy

* **Green State (Optimal):** User within balance; standard Acolyte interaction.
* **Yellow State (Damping):** Drift detected; active intensity modulation or data throttling.
* **Red State (Breach/Shutdown):** Pathological drift; immediate Kill Switch execution.
